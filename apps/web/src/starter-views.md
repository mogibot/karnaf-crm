# Starter Views

## Dashboard
Show:
- leads today
- unanswered now
- hot leads
- payment pending
- SLA risk
- queue counts

## Leads Workspace
Show:
- searchable table/cards
- filters by status, heat, ownership
- recent updates

## Lead Detail
Show:
- lead summary
- transcript timeline
- queue items
- operator actions
- ownership state

## Queue Center
Show:
- pending queue items
- grouped by queue type
- priority and due times
