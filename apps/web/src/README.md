# apps/web/src

This is the starter location for the Karnaf CRM Core operator console frontend.

Recommended first implementation pages/components:
- dashboard
- leads workspace
- lead detail page
- queue center
- system health panel

Recommended first data bindings:
- `/functions/v1/dashboard-summary`
- `/functions/v1/leads-list`
- `/functions/v1/lead-detail?leadId=...`
- `/functions/v1/queue-list`
