# רשימת תיוג להדרכת מפעיל (Mia)

תרגול ב-staging. עבור על כל הסעיפים ביום הראשון. אם נתקעת — קרא ל-[operator-guide.md](operator-guide.md) או למפתח.

## כניסה
- [ ] לחצתי על הקישור ל-staging
- [ ] התחברתי עם המייל שלי
- [ ] שיניתי את הסיסמה הזמנית

## ניווט בסיסי
- [ ] עברתי בין כל הלשוניות (Dashboard / Leads / Queue / Analytics)
- [ ] לחצתי על "?" ליד תפקיד וראיתי את ההרשאות שלי
- [ ] פתחתי תפריט מובייל (אם רלוונטי) ובחנתי שכל הקישורים זמינים

## לידים
- [ ] סיננתי לפי סטטוס "responded"
- [ ] חיפשתי בשם של ליד דמה
- [ ] חיפשתי בתוכן הודעות (toggle "תוכן הודעות") על מילה שמופיעה בשיחה
- [ ] שמרתי תצוגה בשם "החמים שלי"
- [ ] בחנתי שתצוגה השמורה מופיעה גם אחרי refresh

## עמוד ליד
- [ ] פתחתי ליד דמה
- [ ] קראתי את התראנסקריפט
- [ ] שלחתי תשובה ידנית
- [ ] תיעדתי שיחת טלפון (outcome=connected, 5 דק׳)
- [ ] ערכתי את "חסם עיקרי" בסייד-בר
- [ ] לחצתי "סימון כאבוד" → הופיע dialog → "ביטול" — וודאתי שלא קרה כלום
- [ ] לחצתי "סימון כאבוד" → "אישור" — הליד עבר ל-lost

## תור
- [ ] פתחתי `/queue`
- [ ] בחנתי שיש שורות אדומות (SLA נשבר)
- [ ] סגרתי פריט אחד ("סגור")

## ניתוח
- [ ] עברתי על Source performance
- [ ] לחצתי על שורת cohort אחת — נפתחו לידים מהשבוע

## DNC (תרגול קריטי!)
- [ ] בחרתי ליד דמה ללא משמעות
- [ ] לחצתי "DNC" → הופיע dialog → קראתי את האזהרה → אישור
- [ ] וידאתי שהסטטוס שלו עבר ל-do_not_contact ושאי-אפשר לשלוח לו תשובה

## סיום
- [ ] התנתקתי ("יציאה")
- [ ] חזרתי להתחבר וכל ההגדרות נשמרו
