# Karnaf CRM

This repository contains the operating and build specification for Karnaf CRM, a WhatsApp-first, AI-operated CRM and sales system focused on selling the digital program **"הדרך לדירה"**.

## Included files
- `karnaf-crm-full-spec.md` - the full build-ready specification
- `autonomous-crm-operating-model.md` - the operating model and governance layer
- `karnaf-whatsapp-automation-plan.md` - lead capture and WhatsApp automation concept
- `karnaf-whatsapp-sequence.md` - initial WhatsApp follow-up sequence
- `karnaf-gas-apps-script.js` - Google Apps Script webhook starter
- `karnaf_lead_capture.py` - Google Sheets lead-capture helper

## Purpose
Use this repository as the handoff package for design, development, and system implementation.
