# Karnaf CRM - Task Map (2026-04-29)

מטרת המסמך: לפרק את כל הפערים הידועים למשימות נפרדות, עם בעלות ברורה:
- **אלמוג מול מחשב** = דברים שדורשים גישה, אישור, ספק, סוד, או פעולה ידנית חיצונית
- **מוגי עצמאי** = דברים שאני יכול לבצע לבד
- **משותף** = אלמוג פותח/מספק משהו, ואני ממשיך ומסיים

סטטוסי עבודה:
- `OPEN` = עדיין לא נסגר
- `PARTIAL` = יש בסיס, אבל לא סגור
- `BLOCKED-EXTERNAL` = חסום בגלל תלות חיצונית
- `DONE` = סגור
